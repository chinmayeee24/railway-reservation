# railway-reservation
A full-stack web application that allows users to search trains, check seat availability, book tickets, and manage reservations. Built using React, Express, Node.js, and MySQL.
Tech Stack

Frontend: React.js

Backend: Node.js, Express.js

Database: MySQL

API Style: RESTful APIs

 Features

* User registration and login

* Search trains by source and destination

* Check seat availability

* Book and cancel tickets

* View booking history

* Admin can add and manage trains
